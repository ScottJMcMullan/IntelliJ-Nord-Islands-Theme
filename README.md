A theme for IntelliJ IDEs based on the Nord colour palette with support for the new Islands interface.

Based on https://github.com/nordtheme/jetbrains.

## Development

```bash
# Build the plugin
./gradlew buildPlugin

# Run a sandboxed IntelliJ IDEA with the theme installed
./gradlew runIde

# Run a sandboxed PhpStorm with the theme installed
./gradlew runPhpStorm
```
